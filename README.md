    # 📘 Revisão de Fundamentos em Java

## 🚀 Conteúdos abordados

### 🧮 Tipos de Dados
- Inteiros: `byte`, `short`, `int`, `long`  
- Ponto flutuante: `float`, `double`  
- Caracteres: `char`  
- Booleanos.  
- Strings e Arrays.  

### 🔄 Estruturas de Controle
- Laços de repetição: `while`, `do-while`, `for`  
- Condicionais: `if`, `if-else`, `switch`  

### 📚 Arrays e Coleções
- Arrays fixos e inicializados com valores  
- `ArrayList` e suas operações:  
  - `add` → adiciona elementos  
  - `remove` → remove elementos por índice  
  - `get` → acessa elementos por índice  
  - `set` → atualiza elementos por índice  
  - `size` → retorna o número de elementos  
  - `contains` → verifica existência de elemento  
  - `clear` → remove todos os elementos  

### ⚙️ Métodos e Construtores
- Método `main`: ponto de entrada do programa, coordena a execução e delega tarefas para outros métodos e objetos  
- Construtores: métodos especiais que inicializam objetos  
- Criação de métodos (`void` e com retorno)  
- Uso de `this` para referenciar atributos da classe  

### 🔒 Modificadores de Acesso
- `public`, `private`, `protected` e default (package-private)  
- Como controlar a visibilidade de atributos e métodos  
- Reforço de conceitos de encapsulamento  

### 🔄💡 Casting (Conversão de tipos)
- Casting de ampliação (implícito)  
- Casting de estreitamento (explícito)
