Java é uma linguagem de programação e plataforma de computação amplamente utilizada, orientada a objetos e multiplataforma, lançada pela Sun Microsystems em 1995. Sua principal característica é a portabilidade, resumida pelo lema "escreva uma vez, execute em qualquer lugar" (WORA), graças à Java Virtual Machine (JVM). É rápida, segura e robusta, sendo usada para aplicações web, Android, corporativas e Big Data.
A Programação Orientada a Objetos (POO) em Java é um paradigma que organiza o software em torno de "objetos" em vez de funções e lógica. É baseada no modelamento de entidades do mundo real, agrupando dados (atributos) e comportamentos (métodos) em unidades chamadas classes. Java é uma linguagem fortemente orientada a objetos, onde quase tudo interage através da manipulação de objetos.
Java é uma linguagem fortemente tipada e de tipagem estática. Isso significa que toda variável, parâmetro e retorno de método deve ter seu tipo de dado (int, String, etc.) explicitamente declarado e verificado em tempo de compilação, não podendo ser alterado posteriormente, o que proporciona maior segurança e evita erros de tipo em tempo de execução.
A JVM (Java Virtual Machine - Máquina Virtual Java) é um motor de software que fornece um ambiente de execução para aplicações Java, permitindo que o código escrito nessa linguagem rode em qualquer dispositivo ou sistema operacional. Ela é o coração da plataforma Java e a razão pela qual a linguagem consegue ser independente de plataforma, traduzindo o "bytecode" Java (arquivo .class) para a linguagem de máquina específica do hardware em que está rodando.
O Java é independente de plataforma porque seu código compilado, conhecido como bytecode (.class), não é traduzido para código de máquina nativo imediatamente. Ele roda sobre a JVM (Java Virtual Machine), que traduz o bytecode para o sistema operacional específico (Windows, Linux, macOS) no momento da execução.
